## Skill Directory (13 Orchestrator Skills)

| Skill | Description / Domains Covered |
| :--- | :--- |
| `api-contract` | Defines strict request/response data contracts and safety layers (OpenAPI, Zod). Ensures zero untrusted request bodies reach business logic. |
| `backend-orchestrator` | The overarching Architect for Backend Systems, Database Schema, and Performance Optimization. (Includes Backend Architect, Cache Optimizer, DB Expert, etc.) |
| `cost-optimizer` | Reduces cloud and LLM infrastructure costs through token-clipping and tiered service routing. Maximizes efficiency. |
| `data-logic` | Designs immutable data models, DTOs, and complex transformation pipelines. Enforces data integrity. |
| `frontend-experience` | Master orchestrator for UI/UX audits and Frontend debugging. (Includes Flutter Debugging, UX Design) |
| `integrity-sentinel` | The overarching Gatekeeper for security, quality assurance, system audits, and evaluations. |
| `meta-agent-admin` | Governs the AI agent ecosystem, system evolution, context routing, and documentation standards. |
| `project-architect` | Synthesizes complex requirements into technical PRDs and architectural blueprints. Prioritizes Minimum Viable Complexity. |
| `project-operator` | Maintains project deployments, release cycles, and chaos resilience. (Includes CI/CD, Chaos Engineering) |
| `saas-strategist` | Orchestrates SaaS business strategy, viability analysis, and product growth models. |
| `ui-finish` | Applies premium Liquid Glass aesthetics, micro-interactions, and animations to Flutter UIs. |
