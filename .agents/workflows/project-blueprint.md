# Project Blueprint Guide

Use this guide to start the design of a new application or a major feature.

## 0. CONTEXT RETRIEVAL (JIT)
- [ ] Verify Binary Oratory compliance. IF unsure, use `grep_search` on `@agent_protocols.md`.
- [ ] Activate the `@context-manager` skill to map out the necessary systems.

- [ ] **1. Ideation**: Provide a raw description of the application/feature idea.
- [ ] **2. Analysis**: Call the `@project-architect` skill.
- [ ] **3. Drafting**: Use `.agents/templates/BLUEPRINT.template.md` to fill out Chapters 1-7.
- [ ] **4. Refinement**: Load `@analytical_standards.md`. Perform Recursive Grading until achieving a score of 10.
- [ ] **5. Output**: Generate the final blueprint document for user review.

This workflow replaces the old `maxfeatures.md` for more professional name standardization.
