# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2026-03-14
### Added
- Implement Smart Dependency Discovery for automatic script synchronization.
### Changed
- Sanitize repository by removing deprecated root scripts and internal maintenance tools.

## [1.0.1] - 2026-03-14
### Added
- 'Logical Bridge' fallback for Windows environments without Administrator/Developer Mode.
### Changed
- Refactor project initialization to consolidate local soul into plural '.agents' directory.
- Deprecate singular '.agent' directory for architectural consistency.

## [1.0.0] - 2026-03-14
### Added
- Initial release of Portable Brainvibing: AI Agent Infrastructure.
